# AI Agent Builder - ES6

JointJS + Agentic AI Workflow Builder

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

This will start the webpack dev server on port 8080.

## Build for Production

```bash
npm run build
```

This will create a `dist` folder with the bundled application.

## Integration with .NET

The built output in `dist/` is automatically copied to the .NET app's `wwwroot` during the .NET build process.
